Accessing Unsafe Services

Removing services is one of the most difficult steps to reclaim personal privacy. While you should remove the services entirely, you may find that you simply need some services. This guide shows you how to use these services.

Use the Web Version

Many services offer a mobile or desktop app to use their service. While these are convenient, they have more potential to access your data. If a service has a website you should always use it instead of the app. This is due to a concept called sandboxing. Sandboxing is a technique that limits what the service has access to. The browser has a very strong sandbox environment. Unlike desktop or mobile, the browser assumes that a website is potentially dangerous. It severely reduces what a service is able to access. Browsers are also able to use extensions to limit the service further. If your service has a web version, uninstall the mobile/desktop versions.

Choosing a Browser

Much of this information is adopted from the Browser Guide. I highly suggest you read it!

You need to choose a browser with strong privacy and sandboxing abilities. The main options are the Brave Browser and LibreWolf. Both of these are secure by default. However, we will go further. In both browsers, you should set the cookies, cache, and site data to auto-delete. For Brave, this can be done on the Security Page. For Librewolf, this is done automatically.

Using Extensions

You will need several extensions to use a service safely. These are Ublock-Origin, Cookie-Autodelete, and Decentraleyes. Ublock limits what the website is allowed to load. Common ads, trackers, and privacy invasive scripts are blocked. Cookie-Autodelete can be set up to delete cookies when a tab is closed. Decentraleyes is used to serve static CDN content from an alternative source. These three work together to protect your data.

Reducing Dependence

While these tools make a service safer, they cannot fully protect you. Try to reduce the amount of time you use each service. Over time, you may find that you no longer need it.
