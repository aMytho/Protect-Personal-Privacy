Browser Fingerprinting

Below lists all the information that the browser has with your current security settings. If information is missing or not accurate, that is a good thing! Some stats will vary based on extensions and privacy settings.

Displays

Property Value Description
sizeScreenW {{info.sizeScreenW}} The width of the screen in pixels
sizeScreenH {{info.sizeScreenH}} The height of the screen in pixels
sizeInW {{info.sizeInW}} The width of the browser window in pixels
sizeInH {{info.sizeInH}} The height of the browser window in pixels
sizeAvailW {{info.sizeAvailW}} The available width of the screen in pixels
sizeAvailH {{info.sizeAvailH}} The available height of the screen in pixels
scrColorDepth {{info.scrColorDepth}} The bit depth of the color palette for displaying images on the screen
scrPixelDepth {{info.scrPixelDepth}} The bit depth of the screen

Browser Info

Property Value Description
browserName {{info.browserName}} The name of the browser
browserEngine {{info.browserEngine}} The name of the browser's rendering engine
browserVersion1a {{info.browserVersion1a}} The version of the browser
browserVersion1b {{info.browserVersion1b}} The user agent string of the browser
browserLanguage {{info.browserLanguage}} The language of the browser
browserOnline {{info.browserOnline}} Whether the browser is currently online
browserPlatform {{info.browserPlatform}} The platform on which the browser is running
javaEnabled {{info.javaEnabled}} Whether Java is enabled in the browser
dataCookiesEnabled {{info.dataCookiesEnabled}} Whether cookies are enabled in the browser
dataCookies1 {{info.dataCookies1}} The contents of the cookies
dataCookies2 {{info.dataCookies2}} The decoded contents of the cookies
dataStorage {{info.dataStorage}} The contents of the local storage

General

Property Value Description
timeOpened {{info.timeOpened}} The date and time the page was opened
timezone {{info.timezone}} The timezone offset in hours from UTC
pageon {{info.pageon}} The current page's path
referrer {{info.referrer}} The URL of the referring page
previousSites {{info.previousSites}} The number of sites in the browsing history
